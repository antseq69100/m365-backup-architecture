# Restore Procedures – Veeam Backup for Microsoft 365

## Exchange Restore

Open:

```
Veeam Explorer for Exchange
```

Steps:

1. Select organization
2. Browse mailbox
3. Select restore point
4. Restore item or mailbox

Supported restore modes:

```
Restore to original location
Restore to another mailbox
Export to PST
```

---

## OneDrive Restore

Open:

```
Veeam Explorer for OneDrive
```

Steps:

1. Select user
2. Browse restore point
3. Select file or folder
4. Restore or export

---

## SharePoint Restore

Open:

```
Veeam Explorer for SharePoint
```

Steps:

1. Select site
2. Browse document library
3. Select restore point
4. Restore item or export

---

## Teams Restore

Open:

```
Veeam Explorer for Teams
```

Steps:

1. Select team
2. Select channel
3. Select restore point
4. Restore conversation or files
